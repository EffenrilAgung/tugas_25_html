Mengenal Semantic Elements Di HTML5

  • Article

    • Ketika kita ingin membuat layout untuk menampilkan block artikel-ertikel yang ada di website kita, tag yang bisa kita gunakan dengan element header dan footer. Karena tag Article memiliki type block, artinya element ini memiliki type block, artinya element ini memiliki lebar sesuai dengan lebar browser jika kita tidak menentukan lebarnya.

  • Aside

    • Element ini bisa kita gunakan untuk melengkapi layout utama. misalnya kita memiliki halaman blog maka kita bisa menambahkan aside di bagian bawah postingan untuk menjelaskan identitas penulis artikel karena aside memiliki type/sifat block maka lebarnya sesuai page jika kita tidak menentukan lebarnya.
